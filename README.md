## Pokretanje aplikacije

### Backend
1. U terminalu otvoriti direktorij s backendom.
2. Instalirati ovisnosti naredbom npm install
3. Pokrenuti server naredbom nodemon index.js

Backend će biti dostupan na adresi `http://localhost:3000`.

### Frontend
1. U terminalu otvoriti direktorij s frontendom.
2. Instalirati ovisnosti naredbom npm install
3. Pokrenuti aplikaciju naredbom npm run dev

Frontend će biti dostupan na adresi `http://localhost:5173`.
